TAGS EM HTML 5

Estrutura básica do html

        
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
                
</body>
</html>

<!DOCTYPE html>

Significa que o documento está escrito em HTML 5.

<html lang="en">

Signica o idioma que será utilizado, no caso "en" = english.

<html>

O corpo do código, engloba todos os outros elementos do html.

<head>

Contem as informações do documento que serão interpretadas pelo navegador.

<body>

Onde fica o conteúdo do site, os textos imagens, e tudo aquilo que o usuário vai interagir.

<meta>

Metadados, informações sobre os dados do documento.

<link>

Faz referencia a documentos externos, usado para vincular folhas de estilos externos e para definir favicon da pagina.
