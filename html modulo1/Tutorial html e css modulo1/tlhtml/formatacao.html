FORMATAÇÃO EM HTML5

<strong>

Usado para colocar um texto em destaque (negrito).

<em>

Usado para colocar um texto em enfase (italico).

<mark>

Usado para adicionar uma marcação ao texto (marca texto).

<small>

Colocar um texto em tamanho pequeno (letrinha miuda).

<del>

Texto deletado (traçado por cima do texto).

<ins>

Marcação no texto inserido (linha abaixo do texto).

<sup>

Para inserir texto sobrescrito (X20+3).

<sub>

Para inserir texto subescrito (H2O).

<q>

Colocar texto entre aspas (Citações).

<blockquote>

Usado para fazer citações complexas
